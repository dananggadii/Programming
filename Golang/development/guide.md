# Berikut langkah-langkah dalam menjalankan GO

### Check version GO
```bash
go version
```

### Untuk membuat module pada folder 
```bash
go mod init 'nama module'
```

### Compile go file and run
```bash
go build
```

```bash
./compile-file
```


### run go without compile
```bash
go run 'file_name'
```
